# Dashboard Tour

A quick walkthrough of the RantAI Agents dashboard and what each section does.

## Sidebar Navigation

The sidebar on the left is your main navigation. Here's what each menu item does:

<Screenshot src="/images/getting-started/sidebar-menu.png" alt="Sidebar Menu" />

### Main Features

| Menu Item | What It Does |
|---|---|
| **Chat** | Start AI conversations with your assistants |
| **Agent Builder** | Create and configure custom AI assistants |
| **Workflows** | Build visual automation pipelines |
| **Digital Employees** | Autonomous AI agents in Docker containers *(coming soon)* |
| **Live Chat** | Handle customer conversations handed off from AI |
| **Files** | Upload and manage documents & knowledge bases |
| **Marketplace** | Browse and install pre-built assistants, skills, tools, MCP servers, and workflows |

### Chat Section

Click **Chat** to expand the section. You'll see your recent conversations and a button to start a new chat.

<Screenshot src="/images/getting-started/sidebar-chat-section.png" alt="Chat Section" />

### Agent Builder Section

Click **Agent Builder** to see all your assistants. Each assistant has an emoji, name, and description.

<Screenshot src="/images/getting-started/sidebar-agent-section.png" alt="Agent Section" />

### Bottom Navigation

At the bottom of the sidebar you'll find:
- **Settings** — Platform configuration
- **User profile** — Click to access account and logout

<Screenshot src="/images/getting-started/user-menu.png" alt="User Menu" />
