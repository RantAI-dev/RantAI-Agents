# Manage History

All your conversations are automatically saved. You can revisit, search, export, and manage your chat history.

## View Chat History

Your recent conversations appear in the sidebar under the **Chat** section.

<Screenshot src="/images/chat/chat-history-sidebar.png" alt="Chat History Sidebar" />

Each conversation shows:
- **Title** — Auto-generated or custom topic title
- **Assistant emoji** — Which assistant was used
- **Time ago** — How long ago the conversation happened
- **Active indicator** — Colored left border on the current conversation

Click any conversation to reopen it and continue where you left off.

## Edit History & Versions

When you edit a message, the original is preserved. You can toggle between versions using the version indicator.

<Screenshot src="/images/chat/chat-edit-versions.png" alt="Edit Versions" />

- User messages show edit count
- Assistant responses track regeneration versions
- Navigate between versions with arrow buttons

## Export Conversations

Download your conversation for sharing or archiving.

<Screenshot src="/images/chat/chat-export.png" alt="Export Dialog" />

Available formats:
- **Markdown (.md)** — Formatted text with message roles and timestamps
- **JSON (.json)** — Structured data with full metadata

Export includes: conversation title, timestamp, message count, and all user/assistant messages.

## Delete Conversations

Hover over a conversation in the sidebar and click the delete button to remove it.

## Tips

- Conversations are saved automatically — no need to manually save
- Each conversation remembers the assistant and toolbar settings that were used
- Start a new chat to get a fresh conversation without previous context
- Use export to save important conversations before cleanup
