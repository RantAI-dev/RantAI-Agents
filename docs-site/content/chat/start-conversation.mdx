# Start a Conversation

Learn how to start chatting with an AI assistant.

## Open Chat

Click **Chat** in the sidebar. You'll see the chat home screen with a greeting, quick actions, and your recent conversations.

<Screenshot src="/images/chat/chat-empty.png" alt="Chat Home" />

### Quick Actions

The home screen provides shortcuts to:
- **Create Agent** — Jump to the Agent Builder
- **Files** — Open Files management
- **Workflows** — Go to Workflows

### Quick Suggestions

Based on the selected assistant, you'll see suggested prompts as clickable buttons to help you get started.

## Select an Assistant

Click the assistant selector to choose which assistant to talk to.

<Screenshot src="/images/chat/chat-select-assistant.png" alt="Select Assistant" />

Each assistant shows:
- **Emoji & name** — Visual identifier
- **Description** — What the assistant does
- **Knowledge base icon** — Indicates if a knowledge base is attached
- **Default badge** — Marks the default assistant

You can also click **New Assistant** at the bottom to create one directly from the chat.

## Start a New Chat

Click the **New Chat** button in the sidebar, or use the keyboard shortcut **Cmd/Ctrl+B**.

<Screenshot src="/images/chat/chat-new-button.png" alt="New Chat Button" />

## Send a Message

Type your message in the input box and press **Enter** to send. Use **Shift+Enter** for a new line.

<Screenshot src="/images/chat/chat-conversation.png" alt="Active Conversation" />

The assistant will respond in real-time with streaming text. If a knowledge base is attached, responses may include source citations.

<Screenshot src="/images/chat/chat-with-knowledge.png" alt="Chat with Knowledge" />

Sources show the favicon, domain name, and section — click to view the original document.

## Tips

- Each assistant can have different capabilities — try different ones for different tasks
- Quick suggestions change based on the selected assistant
- The chat home shows your recent conversations for easy access
