# MCP Servers

Browse and install Model Context Protocol (MCP) servers for extensible tool discovery.

<Screenshot src="/images/marketplace/marketplace-mcp.png" alt="MCP Tab" />

## What is MCP?

The **Model Context Protocol** is an open standard that allows AI models to discover and use tools dynamically. MCP servers expose tools that assistants can call during conversations — no need to manually configure each tool.

## How It Works

1. **Install an MCP server** from the Marketplace
2. The server appears in **Settings → Agent Config → MCP**
3. **Enable** the server and it auto-discovers available tools
4. **Attach it to an assistant** in the Agent Builder → MCP tab
5. The assistant can now use all tools exposed by that server

## Installing an MCP Server

1. Navigate to **Marketplace** → **MCP** tab
2. Browse or search for a server
3. Click **Install**
4. If the server requires configuration (connection settings, API keys), a config form will appear
5. Once installed, manage it in [Settings → Agent Config → MCP](/docs/settings/agent-config#mcp)
