# Digital Employees

> This feature is currently under development.

Digital Employees are autonomous AI agents that run in isolated Docker containers and can perform tasks independently with different levels of autonomy.

<Screenshot src="/images/digital-employees/preview-list.png" alt="Preview" />

## Planned Capabilities

### Autonomy Levels

| Level | Name | Description |
|---|---|---|
| **L1** | Supervised | Every action requires human approval |
| **L2** | Semi-Autonomous | Routine tasks auto-approved, complex tasks need approval |
| **L3** | Autonomous | Most tasks auto-approved, only high-risk needs approval |
| **L4** | Unrestricted | Full autonomy — no approval required |

### Workspace IDE

Each digital employee has its own isolated workspace with:
- **File Explorer** — Browse and manage the employee's files
- **Terminal** — Execute commands in the employee's environment
- **VNC Remote Desktop** — See exactly what the employee is doing on screen

<Screenshot src="/images/digital-employees/preview-workspace.png" alt="Workspace Preview" />

### Additional Features

- **Triggers** — Start employees manually, on a schedule (cron), or via webhooks
- **Inter-Employee Messaging** — Delegation, task routing, and broadcasts between employees
- **Goal Tracking** — KPIs and counters with auto-tracking from runs
- **Skills from ClawHub** — Install skills from the marketplace
- **20+ Integrations** — Connect to Slack, GitHub, Gmail, and more
