# Integrations

> This feature is part of **Digital Employees** and is currently under development.

Integrations allow digital employees to connect to 20+ external services and interact with the tools your team already uses.

## Planned Integrations

| Service | Type | Capabilities |
|---|---|---|
| **Slack** | Communication | Send messages, read channels, respond to mentions |
| **Gmail** | Email | Send/read emails, search inbox |
| **GitHub** | Development | Create issues, PRs, read repositories |
| **Google Drive** | Storage | Read/write files, search documents |
| **Notion** | Productivity | Create/update pages, search workspace |
| **Linear** | Project Management | Create/update issues, manage projects |
| **Discord** | Communication | Send messages, manage channels |
| **Telegram** | Messaging | Send/receive messages |
| **Salesforce CRM** | CRM | Manage contacts, opportunities, cases |
| **WhatsApp** | Messaging | Send/receive messages via Meta Cloud API |
| **Custom APIs** | Any | Connect any REST API |
| **Custom MCP** | Protocol | Connect any MCP-compatible server |

## How Integrations Will Work

1. **Connect** — Authenticate with the service (OAuth or API key)
2. **Assign** — Grant an employee access to the integration
3. **Use** — The employee can interact with the service during tasks

> **Note:** Assistants in the Agent Builder already support tools and MCP servers. Runtime integrations with credential management are specific to Digital Employees.
