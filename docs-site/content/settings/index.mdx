# Settings

Configure your platform, manage your organization, and control access.

<Screenshot src="/images/settings/settings-overview.png" alt="Settings" />

## Settings Pages

| Page | Tabs | What You Can Configure |
|---|---|---|
| [**General**](/docs/settings/general) | Features, About | Agent Portal toggle, channels config (WhatsApp, Email, Salesforce), system info |
| [**Agent Config**](/docs/settings/agent-config) | Tools, Skills, MCP, Credentials | Manage tools, skills, MCP servers, API keys & credentials |
| [**Analytics**](/docs/settings/analytics) | Memory, Statistics | AI memory management, usage stats, token costs, charts |
| **Organization** | Overview, Members | Organization name, members, roles, invitations *(coming soon)* |
| **Billing** | — | Plan, usage limits, payment method, invoices *(coming soon)* |
