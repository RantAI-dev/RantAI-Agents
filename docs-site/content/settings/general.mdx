# General

Platform feature toggles, channel configuration, and system information.

## Features Tab

<Screenshot src="/images/settings/general-features.png" alt="General Features Tab" />

Enable or disable platform capabilities and configure communication channels. Admin access required.

### Agent Portal

- **Agent Portal** — Enable/disable the customer support queue and live chat. When enabled, the [Live Chat](/docs/live-chat) menu appears in the sidebar.

### Channels

<Screenshot src="/images/settings/general-features-channels.png" alt="Features Channels" />

Configure live agent channels for customer handoff. Each channel can be independently enabled and configured with its own **Enable/Disable** toggle and **Save** button.

| Channel | Description |
|---|---|
| **Portal** | Real-time web chat via WebSocket (always available) |
| **WhatsApp Business** | Send notifications to agents via WhatsApp |
| **Email** | Send confirmation emails to customers via SMTP |
| **Salesforce Messaging** | Handoff to Salesforce embedded chat widget |

See [Live Agent Channels](/docs/channels/live-agent) for the full configuration guide for each channel.

## About Tab

<Screenshot src="/images/settings/general-about.png" alt="About Tab" />

View system information:
- Brand logo and product name
- Version number
- Company website link
- Feature overview grid (AI Chat, Knowledge Base, Live Agent Handoff, Agentic Tools, Multi-Channel, Memory System)
