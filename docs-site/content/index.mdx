# Welcome to RantAI Agents

Enterprise-grade AI agent platform for building and deploying intelligent assistants with RAG capabilities, multi-channel communication, and human-in-the-loop workflows.

## What Can You Do with RantAI Agents?

| Feature | Description |
|---|---|
| [**Chat with AI**](/docs/chat) | Start conversations with intelligent assistants that understand your documents and data |
| [**Build Custom Assistants**](/docs/agent-builder) | Create and configure AI assistants tailored to your specific needs |
| [**Automate Workflows**](/docs/workflows) | Design visual automation pipelines with approval gates and integrations |
| [**Manage Knowledge**](/docs/files) | Upload documents and build a searchable knowledge base powered by RAG |
| [**Connect Integrations**](/docs/integrations) | Link 20+ services like Slack, Gmail, GitHub, Notion, and more |
| [**Deploy Anywhere**](/docs/channels) | Embed chat widgets, connect WhatsApp, or use the web portal |

## Key Features

- **AI Chat** — Converse with assistants powered by leading LLMs via OpenRouter
- **Agent Builder** — Create custom assistants with specific instructions, tools, and knowledge
- **Visual Workflows** — Drag-and-drop automation builder with approval gates
- **Knowledge Base** — Upload PDF, Word, images, and more — AI answers grounded in your documents
- **Marketplace** — Browse and install pre-built assistants, skills, tools, and MCP servers
- **20+ Integrations** — Slack, Gmail, GitHub, Google Drive, Notion, Linear, and more
- **Multi-Channel** — Web portal, embeddable widget, WhatsApp, and email
- **Audit Trail** — Full activity logging with risk scoring for compliance
- **Groups & Teams** — Organize assistants and employees into collaborative teams
