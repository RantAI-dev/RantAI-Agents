# Audit

> This feature is part of **Digital Employees** and is currently under development.

The Audit module provides full activity logging and compliance tracking, primarily for monitoring digital employee actions.

## Planned Capabilities

- **Activity logging** — Every action recorded with who, what, when, and where
- **Risk scoring** — Low, Medium, High, and Critical risk levels
- **IP tracking** — Originating IP address for each action
- **Filtering** — Filter by date range, risk level, action type, and user
- **Export** — Download audit logs for compliance reporting

## Risk Levels

| Level | Examples |
|---|---|
| **Low** | View actions, searches, reads |
| **Medium** | Create and update operations |
| **High** | Delete actions, configuration changes |
| **Critical** | Security-related actions, credential changes |
