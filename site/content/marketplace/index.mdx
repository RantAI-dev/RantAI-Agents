# Marketplace

Browse and install pre-built components to extend your platform — assistants, skills, tools, MCP servers, and workflows.

<Screenshot src="/images/marketplace/marketplace-overview.png" alt="Marketplace Overview" />

## Categories

The Marketplace is organized into 5 tabs:

| Tab | Icon | Description |
|---|---|---|
| **Assistants** | Bot | Pre-built AI assistants ready to deploy |
| **Skills** | Sparkles | Instruction-based capabilities for agents |
| **Tools** | Wrench | Function-calling tools and integrations |
| **MCP** | Plug | Model Context Protocol servers |
| **Workflows** | Workflow | Pre-built automation pipelines |

## Browsing

Each tab provides:
- **Search** — Search by name, description, or tags
- **Category filter** — Filter by category with color-coded checkboxes
- **Sort** — Featured (default), A–Z, or Installed first (skills/tools/MCP only)
- **Stats** — Total available count + installed count

## Item Cards

Every item is displayed as a card with:
- **Icon** — Custom icon or emoji with type-specific color background
- **Display name** and **type badge** (color-coded per type)
- **Description** — Truncated to 2 lines
- **Tags** — First 3 tags shown, "+N" for extras
- **Category** label at the bottom
- **Badges** — "Built-in" (always on) or "Community" where applicable
- **Action button** — Install, Installed/Remove, or Use Template

## Item Detail

Click any card to open the detail dialog showing full description, version, author, all tags, and type-specific content.
