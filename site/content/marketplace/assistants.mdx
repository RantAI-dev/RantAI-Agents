# Assistants

Browse pre-built AI assistants that are ready to deploy.

<Screenshot src="/images/marketplace/marketplace-assistants.png" alt="Assistants Tab" />

## What Are Marketplace Assistants?

These are professionally configured assistant templates with pre-defined system prompts, model settings, tool configurations, and recommended setups.

## How to Use

1. Navigate to **Marketplace** in the sidebar
2. You'll land on the **Assistants** tab (default)
3. Browse or search for an assistant
4. Click **Use Template** on the one you want
5. You'll be taken to the **Agent Builder** with the template pre-loaded
6. Customize the settings as needed and save

> Assistants use the **"Use Template"** action — they create a new assistant in your Agent Builder based on the template, rather than installing a shared component.

## After Using a Template

The new assistant appears in your **Agent Builder** list. You can:

- Use it immediately in **Chat**
- Customize its system prompt, model, and tools
- Attach your own knowledge base
- Rename and rebrand it
