# Files

Files lets you upload documents, organize them into knowledge bases and categories, and enable AI assistants to answer questions using your data via RAG (Retrieval-Augmented Generation).

<Screenshot src="/images/files/files-list.png" alt="Files List" />

## How It Works

1. **Create a knowledge base** — Organize documents by topic or purpose
2. **Upload documents** — Add files in 50+ supported formats
3. **Automatic processing** — Documents are chunked, embedded, and indexed
4. **Attach to assistants** — Connect a knowledge base to any assistant
5. **Get grounded answers** — The assistant cites your documents when responding

> **Note:** Artifacts generated in [Chat](/docs/chat/features#artifacts--files) (HTML pages, React components, 3D scenes, etc.) are also saved here automatically. They appear alongside your uploaded documents with their artifact type icon.

## Supported Formats

| Category | Formats |
|---|---|
| **Documents** | PDF, Word (.docx), PowerPoint (.pptx), RTF, eBook (.epub), OpenDocument (.odt) |
| **Spreadsheets & Data** | Excel (.xlsx), CSV, TSV, JSON, JSONL, OpenDocument Sheet (.ods) |
| **Text & Markup** | Markdown, Plain Text, Log, HTML, XML, YAML, TOML, INI, .env |
| **Code** | Python, TypeScript, JavaScript, Go, Rust, Java, Ruby, PHP, Shell, SQL, Swift, Kotlin, C, C++ |
| **Images (OCR)** | PNG, JPEG, WebP, GIF, HEIC |
| **3D Models** | glTF, glTF Binary (.glb) |

## Files List Page

The list page shows all your documents with:
- **Grid or List view** — Toggle between card view and table view
- **Search** — Find documents by title or category
- **Category filter** — Filter by color-coded categories (multi-select)
- **Sort** — Newest first, Oldest first, or Title A–Z
- **Knowledge base filter** — View documents from a specific group, or all documents
- **Document status** — Indexing (with spinner), chunk count, or chunking failed
