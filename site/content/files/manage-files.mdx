# Manage Files

Organize your documents with knowledge bases, categories, and search.

## Knowledge Bases

Knowledge bases are groups that organize your documents by topic or purpose.

<Screenshot src="/images/files/files-knowledge-bases.png" alt="Knowledge Base List" />

### Create a Knowledge Base

1. Click **New Knowledge Base** in the sidebar
2. Fill in:
   - **Name** — A descriptive name
   - **Description** (optional) — What this knowledge base is for
   - **Color** — Pick from 8 preset colors for visual identification
3. Click **Create**

### Edit or Delete

Use the dropdown menu on the knowledge base header to edit or delete. When deleting, documents are unassigned but not deleted.

### Filter by Knowledge Base

Click a knowledge base in the sidebar to filter the file list. Click **All Documents** to view everything.

## Categories

Categories provide finer organization within knowledge bases.

<Screenshot src="/images/files/files-categories.png" alt="Category Management" />

- **Color-coded** — Each category has a distinct color
- **Multi-select** — Documents can belong to multiple categories
- **System categories** — Some categories are built-in (label cannot be changed)
- **Create inline** — Add new categories directly from the upload or filter dialogs

## Edit a Document

Click the dropdown menu on any document card/row and select **Edit**.

<Screenshot src="/images/files/files-document-edit.png" alt="Edit Document" />

Editable fields:
- **Title** — Update the document name
- **Categories** — Change category assignments
- **Subcategory** — Update subcategory
- **Knowledge Bases** — Move between knowledge base groups

## Search & Filter

### Search
Use the search bar to find documents by title or category name. Search is real-time and case-insensitive.

### Category Filter
Click the **Filter** button to open the category filter. Select multiple categories — documents matching any selected category will be shown.

### Sort
Sort documents by:
- **Newest first** (default)
- **Oldest first**
- **Title A–Z**

### View Mode
Toggle between **Grid** (cards with thumbnails) and **List** (compact table rows).

## Tips

- Keep knowledge bases focused — one per topic area works best
- Remove outdated documents to keep answers accurate
- Use categories consistently across documents for easy filtering
- Check document status to ensure all chunks are indexed
