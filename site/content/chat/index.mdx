# Chat

Chat is the primary way to interact with AI assistants in RantAI Agents. Start conversations, ask questions, generate content, and get answers grounded in your knowledge base.

<Screenshot src="/images/chat/chat-list.png" alt="Chat List" />

## How It Works

1. **Select an assistant** — Choose which AI assistant you want to talk to
2. **Send a message** — Type your question or request
3. **Get a response** — The assistant responds using its configured model, tools, and knowledge
4. **Continue the conversation** — Follow up with more questions in the same session

## Features

- **Multiple assistants** — Switch between different assistants, each with their own personality and capabilities
- **Knowledge-grounded answers** — When a knowledge base is attached, the assistant cites sources from your documents
- **Chat history** — All conversations are saved and can be revisited later
- **Real-time streaming** — Responses appear in real-time as they're generated
- **File attachments** — Upload images, PDFs, and text files directly in the chat
- **Web search** — Enable web search for up-to-date information
- **Code interpreter** — Run code directly in the conversation
- **Canvas / Artifacts** — Generate rich content like HTML pages, React components, diagrams, spreadsheets, and more
- **Tool calling** — Assistants can use 40+ built-in tools during conversations
- **Message threading** — Reply to specific messages for threaded conversations
- **Conversation export** — Download chat history as Markdown or JSON
