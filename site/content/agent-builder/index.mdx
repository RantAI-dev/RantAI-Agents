# Agent Builder

The Agent Builder lets you create and configure custom AI assistants tailored to your specific needs.

<Screenshot src="/images/agent-builder/assistant-list.png" alt="Assistant List" />

## What is an Assistant?

An assistant is a configured AI agent with:

- **System prompt** — Instructions that define the assistant's personality and behavior
- **AI model** — The language model that powers the assistant (via OpenRouter)
- **Tools & Skills** — External capabilities and instruction-based skills
- **Knowledge base** — Documents the assistant can reference when answering
- **Memory** — Context retention across conversations
- **Guard Rails** — Safety rules and content restrictions
- **Integrations** — Connected services like Slack, Gmail, or GitHub

## What You Can Do

- Create new assistants from scratch or use **Generate with AI**
- Configure 12 tabs of settings (model, tools, skills, workflows, MCP, knowledge, memory, guard rails, and more)
- Test assistants directly in a built-in chat
- Deploy via API, widget embed, or multi-channel
